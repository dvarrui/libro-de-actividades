Chapter 13. Time

Table of Contents
SDL_GetTicks — Toma el numero de milisegundos desde que se inicializazó la libreria SDL.
SDL_Delay — Espera un tiempo especificado en milesegundos para despues continuar con la ejecución del programa.
SDL_AddTimer — Añade un timer que espera un cierto tiempo para depues llamar a la función callback.
SDL_RemoveTimer — Elimina un timer que fué añadido con SDL_AddTimer.
SDL_SetTimer — Cambia el intervalo de tiempo de un timer en ejecución e incluso a la función callback que ejecuta.

SDL proporciona varias funciones multiplataforma para tratar el tiempo. Esto proporciona un vía facil para tomar el tiempo actual, esperar un intervalo de tiempo de tiempo mientras se hace algo, y en definitiva SDL proporciona una serie de pequeños mecanismos para controlar el tiempo. Estas funciones permiten dos vías para ejecutar u un objeto cada x milisegundos:

  • Uso de la función callback del timer. Esta función puede causar un mal efecto cuando se ejecuta en hilos separados o es usada en señales de alarma, pero en cambio es muy fácil de implementar.

  • O puedes tomar un tiempo en milisegundos (ejemplo: 30ms) que una vez pasado ejecutan a la función callback
