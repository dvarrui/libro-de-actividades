SDL_JoyButtonEvent

Name

SDL_JoyButtonEvent -- Estructura de evento de botón de joystick.

Definición de la estructura

typedef struct{
  Uint8 type;
  Uint8 which;
  Uint8 button;
  Uint8 state;
} SDL_JoyButtonEvent;

Datos de la estructura

typeSDL_JOYBUTTONDOWN o SDL_JOYBUTTONUP
whichÍndice del dispositivo de joystick
buttonÍndice del botón del joystick
stateSDL_PRESSED or SDL_RELEASED

Descripción

SDL_JoyButtonEvent es un miembro de la unión SDL_Event y se usa cuando se notifica un evento de tipo SDL_JOYBUTTONDOWN o SDL_JOYBUTTONUP.

Un evento de tipo SDL_JOYBUTTONDOWN o SDL_JOYBUTTONUP se produce siempre que el usuario pulsa o despulsa un botón de un joystick. El campo which es el índice del joystick que produjo el evento y el campo button es el índice del botón. (para una explicación más detallada ver la sección de Joystick). state es el estado actual del botón que puede ser SDL_PRESSED o SDL_RELEASED.
