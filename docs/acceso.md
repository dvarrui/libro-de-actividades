# ¿Cómo accedo a los documentos?

Puedes acceder a la documentación de dos formas:

## Acceso ON-LINE

Usando un navegador web, se accede a los documentos de este proyecto alojado en GitHub.

## Acceso OFF-LINE

Puedes tener una copia en tu PC local de todos los ficheros del proyecto:

* Puedes descargar el contenido completo en un fichero ZIP, desde el botón `Download ZIP` de la página web del repositorio GitHub.
* También mediante comandos de Git puedes tener una copia local del proyecto.
    * `git clone https://github.com/dvarrui/libro-de-actividades.git`: Clonar proyecto
    * `git pull`:  Actualizar los fichero en local con las novedades del repositorio.

> **NOTA**: Los documentos en GitHub se modifican y actualizan continuamente.
Por lo que podrán cambiar sus contenidos de un día para otro. La rama `stable` se mantendrá estable durante el curso completo hasta el mes de agosto del siguiente año.
